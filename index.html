DM templates

Four-stage outreach sequence. Copy, personalise, send.

Usage rights — confirm this in every collab

"By accepting the product, you're giving TrueTee permission to repost and use your content for social media and paid ads. We'll always credit you, and you can ask us to remove anything at any time."

A — Cold DMInitial outreach
Hi [Name], [One genuinely specific observation — e.g. "The airport fit reel you posted last week was really clean." Be exact, not generic.] We're building TrueTee — honestly made, everyday tees. No logos, no noise. We're reaching out to a small group of people with a natural everyday style. Simple idea: we send you one of our tees, you wear it and create whatever feels natural. No script, no brand tone. Would you be open to it?
B — Follow-up5 days after no reply
Hey [Name], just a quick follow-up on my message from earlier this week. No pressure at all if you're not interested or it's a busy time. If you are open to it, we'd love to send a tee your way. Just reply and we'll sort it out.
C — BriefAfter they say yes
Amazing — really glad you're in. Some loose ideas if you want a starting point, but do whatever feels natural: • 3 ways you'd style this tee • A day in my life wearing it • Mirror fit check • Honest first impression when it arrives • Coffee run / campus / everyday outfit Format: ideally one reel (30–60 sec). Stories are a bonus. No hard deadline, but within 2 weeks would be great. Quick note: by accepting the tee, you're giving TrueTee permission to repost and use your content for our social and ads. We'll always credit you, and you can ask us to remove anything anytime. What size works for you, and where should we ship?
D — Nudge10 days after delivery
Hey [Name], hope the tee arrived safely! No rush at all — just checking in to see if you had any questions or needed anything from our side. Looking forward to seeing how you style it.

Campaign notes

Settings

Connect Google Sheets, set up AI tools, and deploy your dashboard.

Step 1 — Connect Google Sheets (your database)
1
Create a new Google Sheet
Go to sheets.google.com → New blank spreadsheet → name it "TrueTee UGC Dashboard"
2
Open Apps Script
Extensions → Apps Script → delete all existing code → paste the entire contents of truetee-apps-script.js → Save
3
Deploy as Web App
Deploy → New deployment → Web App → Execute as: Me → Who has access: Anyone → Deploy → Authorise → copy the URL ending in /exec
4
Paste URL below and connect
Every time you update the script code, you must create a new deployment — not edit the old one. The URL will change.
Not connected
Step 2 — AI-powered DMs and content (free)
Use Claude.ai (free) for all AI generations — DM writing, content briefs, hook scripts, and captions. It knows TrueTee's tone and strategy. Just paste your request and copy the output back into your workflow.
Ready-to-use prompts for Claude.ai:
"Write a TrueTee barter DM for @[handle], [X] followers. They posted [describe post]. Under 80 words, genuine, no corporate language."
"Write a TrueTee content brief for a creator whose style is [describe]. Product: [name]."
"Write 3 reel hooks and a caption for TrueTee. Clip: [describe what the creator filmed]."
Step 3 — Deploy to GitHub Pages (free, 3 minutes)
1
Create a GitHub account
Go to github.com → Sign up (free). Pick any username.
2
Create a new repository
Click + → New repository → name it truetee-dashboard → set to Public → tick "Add a README file" → Create repository
3
Upload the dashboard file
In the repository, click Add file → Upload files → drag in truetee-ugc-dashboard.html → rename it to index.html before uploading → Commit changes
4
Enable GitHub Pages
Settings → Pages → Source: Deploy from a branch → Branch: main → folder: / (root) → Save. Wait 60 seconds.
5
Your dashboard is live
Your URL will be https://yourusername.github.io/truetee-dashboard. Open it, paste your Sheets URL and API key in Settings — done. Bookmark it.
After hosting: Enter your Anthropic API key in the AI Setup section above. All three AI tools (DM writer, brief generator, hook writer) will work immediately. The Sheets sync continues working exactly as before.
